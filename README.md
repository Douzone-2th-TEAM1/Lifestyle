## 📂 프로젝트 구조

| 폴더 / 파일 | 설명                  |
| ----------- | --------------------- |
| public      | 루트 디렉토리         |
| images      | 이미지 디렉토리       |
| css         | 각 페이지별 css 파일  |
| js          | 각 페이지별 js 파일   |
| html        | 각 페이지별 html 파일 |

<br/>

## 🏌️‍♂️ 주요 기능

`회원가입`

- 아이디, 비밀번호를 통한 회원가입 기능
- 아이디 중복, 비밀번호 일치 체크 기능
- 로컬 스토리지에 정보 저장

`로그인`

- 아이디, 비밀번호 일치 여부
- 로그인 성공 시 세션 생성

`로그아웃`

- 로그인 세션 제거

`메인 페이지`

- 각 타이틀에 맞게 리스트 페이지로 이동
